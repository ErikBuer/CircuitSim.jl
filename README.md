# CircuitSim.jl

[![](https://img.shields.io/badge/docs-latest-blue.svg)](https://erikbuer.github.io/CircuitSim.jl/dev/)

Julia package for running circuit simulation.

Only qucsator is currently suported.

The package currently expects qucsator to be preinstalled.


## Installation

The package is not registered. Install from repo.

```julia
] add https://github.com/ErikBuer/CircuitSim.jl
```
